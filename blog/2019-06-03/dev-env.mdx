## 요약

이 문서는 React 프로젝트 개발을 위한 로컬 컴퓨터 작업 환경을 설정하는 방법을 가이드 합니다.

- 공통
  - 개발 도구
  - 공개 SW 선정 분석자료
- 프론트엔드
  - 개발환경 설정 가이드
    - 로컬 PC 개발환경 설치 프로그램, 확장 프로그램, 설정 파일 등
    - node, git, yarn, VS Code, 크롬 브라우저, 터미널, VS Code 설정, ESLint 설정, Prettier 설정
  - UI 프레임워크 연계 시스템 이용 환경 설정 방법 가이드
    - Nexus, Bitbucket 등 연동 방법
  - 정상 설정 확인 방법

## 설치 프로그램 소개

React 개발을 위해 기본적으로 필요한 항목은 다음과 같습니다.

- OS **Windows 10 (x64)**: Windows, macOS, Linux 등 OS에서 개발 가능합니다. 다만 추후 Gatsby 개발 및 빌드 시 Windows 32비트의 경우 **Node x64를 요구**하는 플러그인(gatsby-plugin-sharp) 사용에 제약이 있으므로 64비트 설치를 권장합니다.
- 자바스크립트 런타임 **Node.js** : Webpack 과 Babel 같은 도구들이 자바스크립트 런타임인 Node.js 를 기반으로 만들어져있습니다. 그렇기에 해당 도구들을 사용하기 위해서 Node.js 를 설치합니다.
- 패키지 매니저 **npm** : npm 은 Node.js 를 설치하게 될 때 함께 설치되는 도구 입니다. 프로젝트에서 사용되는 라이브러리를 설치하고 해당 라이브러리들의 버전 관리를 하게 될 때 사용하죠.
- 버전 관리 시스템 **Git**
- 코드 에디터 **Vidual Studio Code** (이하 VS Code)
- 브라우저 **크롬**

### 다운로드 링크

각각의 설치 파일을 다음 링크를 통해 다운 받습니다.

- [Node.js](https://nodejs.org/ko/) : LTS 버전을 다운로드 받습니다. 윈도우의 경우 Windows (x64)용 **node-v10.16.0-x64**을 다운로드 받습니다.
- **npm** : npm은 Node.js 설치 시 자동으로 설치 됩니다.
- [Git](https://git-scm.com/downloads) : Windows를 선택하면 자동으로 **Git-2.22.0-64-bit** 프로그램이 다운로드 됩니다.
- [VS Code](https://code.visualstudio.com/#alt-downloads)
- [크롬](https://www.google.com/intl/ko/chrome/)
