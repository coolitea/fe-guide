## 설계서

- 다국어 지원

### 파일 구조

- 소규모
- 대규모

## 공통 기능

- 알림, 비동기처리, 입력폼, 팝업, 데이블, 데이터 그리드, 차트, 코드 관리 등
- 로그인/로그아웃, 인증, 서명, 사용자 설정, 검색, 화면보호, 세션, 쿠키, 다국어 지원, 테마, 메뉴 즐겨찾기 등 사용자 관련 기능
- 인증, 서명, 출력 등 솔루션 연계 기능
- (대용량)파일 업로드/다운로드, 엑셀파일 업로드 후 화면 렌더링 등 파일 및 통신관련 기능
- 결재(신청, 승인, 반려) 등
- Service Worker 등 프로그레시브 웹앱(PWA) 지원 기능

## 공통 컴포넌트

- 아이콘, 폼, 자동완성 폼, 버튼, 드롭다운, 달력, 팝업 알림, 이미지 슬라이더, 반응형 네비게이션 메뉴, 테이블, 인피니트 스크롤링 리스트, 데이터 그리드, 파일 업로드, 트리, 진행상태 표시 등

## 개발 관련 기능

- Node, yarn, git, VS Code, ESLint, Prettier, webpack 등

  - 크롬 개발자도구, Lighthouse, React Developer Tools, Redux DevTools
  - JEST

## 배포 관련 기능
